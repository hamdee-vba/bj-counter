# BJ Counter PWA

Aplikasi perhitungan fisik jaminan kredit — versi Progressive Web App  
URL: https://hamdee-vba.github.io/bj-counter  

**# Install di Android:**
Buka URL di browser Chrome → ketuk ikon ⋮ → **"Add to Home Screen → Install"**

**# Install di iOS (Safari):**
Buka URL di browser Safari → ketuk ikon Share → **"Add to Home Screen"**

## Catatan
- App bekerja **offline** setelah pertama kali dibuka (Service Worker cache)
- Data sesi tersimpan di **localStorage** browser
- Tidak ada data yang dikirim ke server
